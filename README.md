# Gerenciador de Tarefas em Java

Projeto desenvolvido em Java que simula um sistema simples de gerenciamento de tarefas (To-Do List) com interação via terminal.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Remover tarefas
- Menu interativo

## Tecnologias

- Java
- Estrutura de dados (ArrayList)
- Lógica de programação
- Laços de repetição (for, do-while)

## Objetivo

Praticar conceitos fundamentais de programação, como manipulação de listas, estruturas de repetição e interação com o usuário.
